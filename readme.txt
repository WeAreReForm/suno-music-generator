Générateur de musique IA avec Suno via SunoAPI.org

== Description ==

Suno Music Generator vous permet de créer des chansons personnalisées avec l'intelligence artificielle directement depuis WordPress.

= Fonctionnalités =

* Génération de musique par IA
* Interface simple et intuitive
* Support de différents styles musicaux
* Lecteur audio intégré
* Historique des créations

= Shortcodes =

* `[suno_form]` - Formulaire de génération
* `[suno_player]` - Liste des musiques
* `[suno_test]` - Test de connexion (admin)

== Installation ==

1. Téléchargez le plugin
2. Uploadez dans `/wp-content/plugins/`
3. Activez dans WordPress
4. Configurez votre clé API dans Réglages > Suno Music
5. Utilisez les shortcodes sur vos pages

== Configuration ==

1. Obtenez une clé API sur https://sunoapi.org
2. Ajoutez la clé dans Réglages > Suno Music
3. Testez la connexion
4. Commencez à générer !

== Frequently Asked Questions ==

= Où obtenir une clé API ? =

Rendez-vous sur https://sunoapi.org pour créer un compte et obtenir votre clé.

= Combien coûte la génération ? =

Cela dépend de votre plan sur SunoAPI.org. Consultez leur site pour les tarifs.

= Les shortcodes ne fonctionnent pas ? =

Vérifiez que :
1. Le plugin est activé
2. La clé API est configurée
3. Vous avez des crédits sur SunoAPI.org
