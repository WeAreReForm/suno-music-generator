Générateur de musique IA avec l'API officielle Suno

== Description ==

Ce plugin permet de générer des chansons personnalisées en utilisant l'intelligence artificielle de Suno directement depuis WordPress.

= Fonctionnalités =

* Génération de musique via formulaire simple
* Support de l'API officielle Suno
* Interface d'administration intuitive
* Historique des générations
* Test de connexion intégré

= Utilisation =

1. Configurez votre token dans Réglages → Suno Music
2. Ajoutez [suno_form] sur une page
3. Générez vos chansons !

== Installation ==

1. Téléchargez le plugin
2. Décompressez dans /wp-content/plugins/
3. Activez le plugin
4. Configurez votre token Suno
5. Utilisez les shortcodes [suno_form] et [suno_test]

== Frequently Asked Questions ==

= Comment obtenir un token Suno ? =

Connectez-vous sur Suno.com, ouvrez les outils développeur (F12), allez dans Application → Cookies et copiez la valeur de __session.

= Le token expire-t-il ? =

Oui, les tokens expirent généralement après 24-48 heures. Vous devrez le renouveler périodiquement.
